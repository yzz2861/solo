# 图书馆期末清账管理系统 - 示例数据说明

本目录提供 3 份示例数据文件，可直接在系统中导入体验完整流程：

## 数据导入顺序（任选顺序均可，系统按借阅单号自动关联）

1. `borrow_records.csv` - 借阅数据（10 条）
   包含在借、已逾期、已归还等多种情况。

2. `return_records.json` - 还书补录数据（6 条）
   补录部分借阅单的归还日期，用于修正逾期状态。

3. `waiver_applications.json` - 减免申请数据（4 条）
   - WV20260002（BRW20260003）：正常申请
   - WV20260003（BRW20260007）：**申请人不一致** + **减免金额超限**（故意构造的异常，用于测试异常检测）
   - 其余为正常申请

## 推荐体验流程

1. 进入「数据导入」页，依次上传三份文件
2. 查看「借阅单列表」，使用筛选器查看「有异常」的记录
3. 点击 BRW20260002 等进入详情页，填写复核意见并选择减免结论
4. 进入「复核工作台」批量勾选多条记录，一键批量复核
5. 进入「报告导出」页配置字段，预览并导出 CSV 报告
