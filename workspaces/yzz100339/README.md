# 社媒素材发版清单工具

扫描活动目录，按渠道检查素材完整性，生成人类可读的发版清单。

## 功能特性

- **按渠道完整性检查**：自动识别微信/抖音/小红书/微博/B站/快手 6 个渠道
- **海报尺寸校验**：读取图片实际尺寸，与渠道要求尺寸规则对比
- **图片有效性检测**：识别空文件(0B)、损坏文件、无法解析的图片
- **文件名问题检测**：空格、中文括号、版本号冲突、重复终版
- **差异追踪**：对比上次扫描，展示新增/删除/替换的素材
- **人类可读报告**：清晰的行动清单，告知缺什么、谁补什么

## 渠道尺寸规则

| 渠道 | 海报尺寸 | 封面尺寸 |
|------|----------|----------|
| 微信 | 1080x1440, 1080x1080, 1080x1920 | 900x500 |
| 抖音 | 1080x1920, 1080x1080 | 1080x1920 |
| 小红书 | 1080x1440, 1080x1920 | 1080x1440 |
| 微博 | 900x1200, 1080x1080, 1080x1920 | 900x500 |
| B站 | 1146x717, 1080x1920 | 1146x717 |
| 快手 | 1080x1920, 1080x1080 | 1080x1920 |

## 安装

```bash
# 安装依赖
pip install -r requirements.txt

# 或使用 pip3
pip3 install -r requirements.txt
```

## 使用方法

```bash
# 基础用法：扫描活动目录，输出清单到终端
python3 media_release_checklist.py /path/to/活动目录

# 输出到文件
python3 media_release_checklist.py /path/to/活动目录 --output 清单.txt

# 不保存快照（跳过差异对比）
python3 media_release_checklist.py /path/to/活动目录 --no-snapshot

# 自定义快照目录
python3 media_release_checklist.py /path/to/活动目录 --snapshot-dir /tmp/snapshots
```

## 目录结构约定

支持嵌套或扁平的目录结构：

```
活动目录/
  海报/     — 图片素材 (.jpg/.png/.psd/.ai/.webp)
  视频/     — 视频文件 (.mp4/.mov) 及封面图
  文案/     — 文案 (.txt/.md/.doc)
  落地页/   — 链接 (.url/.txt)
  审批/     — 审批备注 (.txt/.md/.pdf)
```

## 文件名编码规则

`{渠道}_{类型}_{版本}.{ext}`

示例：
- `微信_海报_v2_final.png`
- `抖音_封面（终版）.jpg`
