## 数据设计覆盖场景

### 1. 客户换手机号 / 多会员号合并
- **VIP1007 郑慧**：手机号为 139 开头，与其他 138 开头区分，独立客户
- 所有客户各有一个会员号，可通过追加数据测试多会员号场景

### 2. 体验卡 vs 正价项目
- 含"体验卡"关键词的项目：面部护理-体验卡、肩颈按摩-体验卡、精油SPA-体验卡
- 正价项目：面部护理-正价、全身SPA-正价、美白护理-正价
- 套餐类：抗衰套餐、脱毛套餐

## 使用方式

1. 启动应用后进入 **数据导入** 页面
2. 点击 **下载示例CSV** 按钮，或手动从 `sample-data/` 目录选择文件
3. 依次将 4 个 CSV 文件拖拽或上传到对应卡片
4. 系统将自动识别字段映射，如识别正确直接显示 ✓
5. 如需调整映射，点击卡片右上角 ⚙️ 图标配置
6. 点击 **合并导入数据** → **开始分析**
7. 切换到 **图表看板** 查看顾问差异、流失分布
8. 切换到 **流失名单** 安排回访，验证排除管理
