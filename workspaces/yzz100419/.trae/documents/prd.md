## 1. 产品概述

美容院复购流失图——帮助美容院老板识别即将流失的顾客、对比顾问绩效，并让店长精准安排回访。系统整合会员消费、预约、顾问跟进备注和投诉记录四类数据源，按项目和顾问维度计算到店间隔、复购率和流失风险，生成可视化图表和可操作名单，同时保护敏感备注信息、防止不当促销打扰。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 老板 | 查看全局图表看板（顾问差异、项目复购率、流失趋势），不可查看客户敏感备注 |
| 店长 | 查看流失名单、安排回访、管理排除列表、查看敏感备注、导出回访清单 |
| 顾问 | 仅查看自己负责的客户跟进状态，不可查看其他顾问数据 |

### 2.2 功能模块

1. **数据导入页**：上传消费表、预约表、顾问备注、投诉记录四类CSV，配置字段映射，预览合并结果
2. **图表看板页**：顾问复购率对比、项目流失分布、到店间隔趋势、流失风险热力图
3. **流失名单页**：可能流失客户清单、回访任务分配、排除管理、敏感信息权限控制
4. **客户详情弹窗**：单客户消费轨迹、预约历史、顾问跟进时间线、投诉记录

## 3. 核心流程

### 3.1 数据导入与身份合并流程

1. 用户分别上传四类CSV文件
2. 系统解析字段，用户确认映射关系
3. 系统自动匹配：手机号→会员号→姓名组合，识别同一人
4. 用户确认或调整合并结果
5. 系统标记体验卡/正价项目，区分计算权重
6. 导入完成，进入分析阶段

### 3.2 流失分析流程

1. 基于消费记录计算每位客户最近到店日期与历史平均间隔
2. 按项目维度统计复购率（体验卡与正价分开计算）
3. 结合预约爽约率、顾问跟进频率、投诉记录综合评分
4. 生成流失风险等级：高（红色）、中（橙色）、低（黄色）、安全（绿色）
5. 投诉后未处理客户自动标记"暂缓联系"

### 3.3 回访管理流程

1. 店长查看流失名单，按顾问/项目/风险等级筛选
2. 将客户分配给对应顾问回访，设置截止日期
3. 标记排除客户（如刚投诉未处理），禁止群发促销
4. 导出回访清单（可选脱敏），分配给顾问执行

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深玫瑰金 (#B76E79) + 暖灰底 (#F5F0EB)，传达美容行业的高端温暖感
- **辅助色**：风险等级色阶（红 #DC2626 / 橙 #F59E0B / 黄 #EAB308 / 绿 #22C55E）
- **按钮风格**：圆角胶囊按钮，微妙阴影，hover时轻微上浮
- **字体**：标题用 Playfair Display（优雅衬线），正文用 Noto Sans SC（中文适配）
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式模块布局
- **图标**：lucide-react 线性图标，1.5px 描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 数据导入页 | 四表上传 | 四个拖拽上传卡片，每个显示文件名和行数，映射弹窗 |
| 数据导入页 | 身份合并 | 左侧原始数据表，右侧合并结果，连线动画标识匹配 |
| 数据导入页 | 项目标记 | 标签式选择器，体验卡淡色、正价深色区分 |
| 图表看板页 | 顾问复购率 | 横向柱状图，异常值红色高亮，hover显示详细数据 |
| 图表看板页 | 项目流失分布 | 环形图，中心显示总流失数，扇区可点击筛选 |
| 图表看板页 | 到店间隔趋势 | 折线图，双Y轴（间隔天数/客户数），区域填充 |
| 图表看板页 | 流失风险热力图 | 网格矩阵，颜色深浅表示风险等级，点击进入名单 |
| 流失名单页 | 流失客户清单 | 表格+风险标签，顶部筛选条，行可展开详情 |
| 流失名单页 | 回访分配 | 拖拽分配卡片，日历选择截止日期 |
| 流失名单页 | 排除管理 | 侧滑面板，排除原因选择，一键恢复 |
| 客户详情弹窗 | 时间线 | 垂直时间线，图标区分消费/预约/跟进/投诉 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度1280px
- 图表看板在平板端改为纵向堆叠
- 流失名单表在窄屏隐藏次要列，提供展开详情

### 4.4 隐私与权限设计

- 顾问备注字段在老板视图完全隐藏
- 店长视图显示脱敏备注（如"客户有皮肤敏感问题"而非原始措辞）
- 导出清单时可选择"脱敏模式"
- 排除列表客户在所有群发功能中自动过滤
