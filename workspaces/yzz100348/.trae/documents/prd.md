## 1. 产品概述

输液泵剂量核对器是一款面向病区护士的临床辅助工具，用于将医嘱剂量、药液浓度、总量、患者体重和计划输液时间快速换算为 mL/h 泵速，并逐步展示换算过程，降低夜班小数点误读风险。目标用户为病区护士、安全员和质控人员。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 护士 | 直接使用 | 输入参数、查看计算结果、确认并保存记录、打印交班内容 |
| 安全员 | 直接使用 | 审核计算结果、查看质控记录 |
| 质控人员 | 直接使用 | 查询历史记录、追溯原始输入与计算路径 |

### 2.2 功能模块

1. **剂量换算页面**：参数输入、实时计算、换算步骤展示、安全提示
2. **记录管理页面**：历史记录列表、记录详情查看、打印与导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 剂量换算页 | 参数输入区 | 输入医嘱剂量（含单位选择）、药液浓度、总量、体重、计划时间 |
| 剂量换算页 | 计算结果区 | 显示建议泵速 mL/h、体重剂量 mg/kg/min 和 μg/kg/h |
| 剂量换算页 | 换算步骤区 | 逐步展示从医嘱到 mL/h 的完整换算链 |
| 剂量换算页 | 安全提示区 | 体重缺失、浓度为零、结果超范围时醒目警示 |
| 剂量换算页 | 确认保存区 | 护士复核确认、保存记录到本地存储 |
| 记录管理页 | 记录列表区 | 按时间倒序展示历史核对记录 |
| 记录管理页 | 交班打印区 | 精简格式打印，仅含关键结果 |
| 记录管理页 | 质控详情区 | 完整原始输入和计算路径，供质控追查 |

## 3. 核心流程

护士打开核对器 → 输入医嘱剂量、药液浓度、总量、体重和计划时间 → 系统实时计算泵速并展示换算步骤 → 系统自动检测异常（体重缺失/浓度为零/超范围）并醒目提示 → 护士复核确认 → 保存记录 → 可选打印交班内容或查看质控详情

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：医疗蓝（#0F6CBD）+ 安全橙（#D83B01）作为警示色
- 辅助色：浅灰底（#F5F5F5）、白卡片（#FFFFFF）
- 按钮风格：圆角（8px）、主操作按钮实心蓝、次操作按钮描边
- 字体：系统字体为主，数值显示使用等宽字体（JetBrains Mono）
- 布局风格：卡片式布局、顶部导航、表单与结果左右/上下分区
- 图标风格：线性图标（Lucide）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 剂量换算页 | 参数输入区 | 白色卡片、表单输入、下拉单位选择、即时校验 |
| 剂量换算页 | 计算结果区 | 大号等宽数值、单位标签、安全/危险颜色区分 |
| 剂量换算页 | 换算步骤区 | 步骤卡片、箭头连接、公式展示 |
| 剂量换算页 | 安全提示区 | 红色/橙色横幅、图标+文字、固定可见 |
| 剂量换算页 | 确认保存区 | 确认按钮、操作人签名输入、时间戳 |
| 记录管理页 | 记录列表区 | 时间线布局、卡片摘要、筛选排序 |
| 记录管理页 | 交班打印区 | 打印预览、精简表格 |
| 记录管理页 | 质控详情区 | 折叠面板、完整计算路径展示 |

### 4.3 响应式设计

- 桌面优先设计，适配病区工作电脑和平板
- 移动端适配：表单纵向排列、结果区堆叠显示
- 触控优化：输入框和按钮最小 44px 触控区域
