# 房间混响估算器 - 产品需求文档 (PRD)

## 1. 产品概览

### 1.1 产品定位
面向小录音棚装修场景的混响时间估算与材料方案对比工具，服务两类用户：
- **老板/决策者**：需要通俗的建议、材料面积预估、预算核对
- **声学师/技术顾问**：需要完整公式、参数、不确定项说明

### 1.2 核心价值
在装修前快速估算不同频段混响时间，对比多套吸音材料方案，避免"太闷"（低频混响过长）或"太干"（高频混响过短），同时提供材料面积与预算参考。

---

## 2. 功能需求

### 2.1 房间基本信息输入
| 字段 | 类型 | 说明 |
|------|------|------|
| 房间名称 | 文本 | 方案标识 |
| 长 / 宽 / 高 | 数字 | 支持米 (m) 和 英尺 (ft) 切换 |
| 单位制式 | 单选 | 公制 (m) / 英制 (ft) |

### 2.2 六面材料选择（墙×4 + 地 + 顶）
每面独立选择：
- 材料类型下拉（含常见吸音材料，如石膏板、矿棉板、聚酯纤维板、布艺软包、穿孔板、木地板、地毯等）
- 各频段吸声系数（α₁₂₅, α₂₅₀, α₅₀₀, α₁ₖ, α₂ₖ, α₄ₖ）自动填充
- 支持自定义吸声系数
- 显示材料单价（元/㎡，可编辑）

### 2.3 附加吸声体
- **窗帘**：面积 + 材质选择（厚窗帘/薄窗帘/百叶帘）
- **家具**：件数 + 类型（沙发/桌椅/机柜/乐器等），每件估算吸声面积

### 2.4 目标用途
预设用途，提供建议混响时间范围：
- 语音录音棚（播客/配音）：RT₆₀ ≈ 0.3 ~ 0.5s
- 音乐录音棚（独唱/小乐队）：RT₆₀ ≈ 0.4 ~ 0.8s
- 混音室：RT₆₀ ≈ 0.2 ~ 0.4s
- 试听室/HiFi 室：RT₆₀ ≈ 0.3 ~ 0.6s
- 自定义：用户自行输入目标 RT₆₀

### 2.5 方案管理
- 创建/保存/删除方案（本地存储 localStorage）
- 最多支持 5 套方案并排对比
- 一键复制当前方案作为新方案

### 2.6 混响时间计算（核心）
**算法**：
- 体积 V < 50m³：Eyring-Norris 公式（小房间修正）
- 体积 V ≥ 50m³：Sabine 公式
- 输出 6 个倍频程频段（125Hz ~ 4kHz）的 RT₆₀
- 考虑空气吸声修正（>1kHz）

### 2.7 异常与提示
- ⚠️ **单位混用提示**：输入过程中检测到公制/英制数值不合理
- ⚠️ **吸声系数缺失**：所选材料某些频段系数为 0 或未定义
- ⚠️ **体积极小**：房间体积 < 10m³，提示模型可信度降低
- ⚠️ **家具面积超限**：附加吸声体总面积超过对应墙面面积的 80%
- ⚠️ **结果异常**：某频段 RT₆₀ > 3s 或 < 0.1s

### 2.8 报告输出（双版本）

#### 🧑‍💼 老板版（通俗建议）
- 房间概况（体积、总表面积）
- 各频段混响时间结果 + 色彩标签（偏干✅/适中⚠️/偏闷❌）
- 一句话结论（如"低频偏闷，建议增加 15㎡ 低频陷阱"）
- 建议调整方向（加/减哪类材料、大概多少面积）
- **材料清单与预算表**（每面墙/每种材料的面积 × 单价 = 小计，总价）
- 与目标用途的对比柱状图

#### 🎓 声学师版（专业参数）
- 完整计算过程：Sabine/Eyring 公式展开
- 各面表面积、各材料吸声量 A = Σ(Sᵢ × αᵢ) 明细
- 空气吸声修正值 4mV 列表
- 参数来源（材料系数引用标准/厂家数据）
- **不确定项标注**：
  - 材料系数容差 ±15%
  - 家具吸声估算 ±30%
  - 扩散体/不规则面的模型简化
- 与标准曲线（如 ITU-R BS.1116）的对比图

### 2.9 可视化
- 频段 RT₆₀ 曲线图（125Hz ~ 4kHz），多条方案曲线叠加对比
- 目标用途建议范围阴影带
- 预算构成饼图
- 房间 3D 简易示意图（展示六面分配）

## 3. 非功能需求

### 3.1 性能
- 纯前端实现，计算延迟 < 50ms
- 无后端依赖，离线可用
- 支持移动端自适应

### 3.2 数据持久化
- 方案数据保存在 localStorage，Key: `reverb_estimator_schemes_v1`
- 导出方案为 JSON 文件

### 3.3 兼容性
- 支持现代浏览器（Chrome / Edge / Safari / Firefox 最近两个大版本）

---

## 4. 用户故事

1. 作为老板，我输入房间尺寸后，立刻看到大概需要多少平方米材料，方便和装修队谈价。
2. 作为老板，我希望对比 3 套方案的价格和声学效果，选性价比最高的。
3. 作为声学师，我需要保留计算公式和参数来源，写报告时可以直接引用。
4. 作为用户，选错单位（把英尺当米输入）时能及时收到提醒。
5. 作为用户，家具堆得多时系统提示"吸声可能过量"，避免花冤枉钱。

## 5. 设计风格参考

- **色调**：深色专业风 + 暖橙高亮（录音棚控制台美学）
- **字体**：标题用 Orbitron/科技感字体，正文用清晰无衬线
- **交互**：滑块实时反馈、频段进度条渐变色、方案卡片可拖拽排序
