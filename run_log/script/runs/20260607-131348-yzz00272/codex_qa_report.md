- 阻断问题:
  - 幂等键粒度过粗，导致同批次不同明细请求互相覆盖，核心风险判定和审计结果错误。
- 建议:
  - 幂等键应包含可区分请求内容的稳定摘要，或要求显式 idempotency key；仅完全相同请求才返回缓存。
